# grainDay
Shiny app for calculating sediment types
This application is a fun project which might be of use to geologists and ecologists.


Add percentages of grain fraction in your sediment sample, to find their classification


Based on Shephard scale:

Gravel: >=2.0 mm
Sand:  <2>=0.062
Silt: <0.062>=0.004
Clay: <0.004

###

The app draws a ternary diagram of the sediment composition.
